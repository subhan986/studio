# **App Name**: PortfolioPilot

## Core Features:

- Project Showcase: Display a grid of personal projects with titles, brief descriptions, and preview images.
- About Me: Include a brief 'About Me' section with a professional headshot and a concise biography highlighting skills and experience.
- Contact Information: Provide contact information and links to professional profiles (e.g., LinkedIn, GitHub, etc.)
- Skills Section: A section that dynamically lists skills, potentially with visual indicators of proficiency.
- Project Description Generator: AI-powered tool to generate personalized project descriptions, enhancing the portfolio with well-articulated narratives.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), evoking trust and professionalism.
- Background color: Light gray (#F5F5F5), providing a clean, neutral backdrop.
- Accent color: Teal (#009688), used sparingly for call-to-action buttons and interactive elements to draw attention.
- Clean and modern sans-serif fonts, such as 'Helvetica Neue' or 'Arial', for readability.
- Use a clean and structured layout with ample white space to ensure readability and visual appeal.
- Subtle transitions and hover effects to enhance user engagement without being distracting.
- Use a consistent and minimalist icon set to represent different skills and sections.
